学成在线

前期准备

项目目录

code
├── images
├── uploads
├── css
│   ├── base.css
│   └── index.css
└── index.html
  • images：存放固定使用的图片素材，例如：logo、样式修饰图等等
  • uploads：存放非固定使用的图片素材，例如：商品图、宣传图需要上传的图片
  • css/base.css：基础公共样式，例如：清除默认样式、设置网页基本样式
  • css/index.css：首页CSS样式
  • index.html：首页HTML文件

制作思路

  1. 布局思路

    先整体再局部，从外到内，从上到下，从左到右

  2. CSS实现思路

    1. 画盒子，调整盒子范围 -> 宽高背景色
    2. 调整盒子位置 -> flex布局、内外编剧
    3. 控制图片、文字内容样式
